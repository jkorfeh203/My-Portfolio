# John Portfolio

A personal portfolio website built with React and Vite, featuring a rich set of interactive components and a polished UI.

## Sections

- **Hero** — Landing section with typewriter animation
- **About** — Personal introduction
- **Research** — Research work with modal detail view
- **Skills** — Visual skill gauges
- **Experience** — Timeline-based experience display
- **Achievements** — Notable accomplishments
- **Testimonials** — References and recommendations
- **Gallery** — Photo gallery with slideshow modal
- **Contact** — Contact form / info

## Tech Stack

- [React 18](https://react.dev/)
- [Vite 5](https://vitejs.dev/)
- [Lucide React](https://lucide.dev/) — icons

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

The project is configured for deployment on [Vercel](https://vercel.com) via `vercel.json`, which includes security headers (CSP, X-Frame-Options, Referrer-Policy, etc.).

To deploy:
1. Push the repo to GitHub
2. Import the project on vercel.com
3. Vercel auto-detects Vite — click **Deploy**
